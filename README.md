# TripleTen — Biblioteca Triple Peaks

**Descripción:**
- es un sitio web estático que presenta la "Biblioteca Triple Peaks":
	información sobre libros recomendados, eventos y cómo hacerse miembro.  - 

**Características:**
- Diseño accesible (HTML + CSS).
- Secciones principales: `Los preferidos de nuestro staff`, `Eventos`,
	`Conviértete en miembro` y `Contactos`.

**Estructura del proyecto:**
- `index.html` — Página principal estática.
- `styles/` — Hoja(s) de estilo. Principal: `styles/style.css`.
- `vendor/normalize.css` — Reset/normalización de estilos.
- `images/` — Imágenes y assets usados en la web.
